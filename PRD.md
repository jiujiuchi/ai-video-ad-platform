# VidMotion 产品需求文档 (PRD)

## 1. 产品定位

VidMotion 是一个面向中小广告主的 AI 视频广告生成工具。核心价值主张：**让没有视频制作能力和 prompt 工程经验的店主，也能 3 分钟内生成可投放的视频广告。**

## 2. 目标用户

### 主要用户：中小广告主（电商店主、本地商户）

- **痛点**：视频广告 ECPM 高于图片，但制作视频成本高（单条 2000-5000 元）、周期长（3-5 人天）
- **能力特征**：大多数人不会写 AI prompt，没有视频剪辑技能，只有商品图片素材
- **核心需求**：便宜、快、能用——不需要电影级的视频，只要能投放到开屏/Banner/信息流就行

### 次要用户：品牌客户

- **痛点**：对创意内容的增删改有严格把控要求
- **产品策略**：V1 阶段暂不覆盖品牌客户的高定制需求，通过「模板 + 轻微运动」控制生成内容不偏离原始素材

## 3. 核心场景

| 场景 | 用户行为 | 产品响应 |
|------|----------|----------|
| 零基础首次使用 | 打开产品，不知从何下手 | 首页展示模板市场，展示成功案例和 CTR 数据，点击「做同款」直接跳转 |
| 日常批量生成 | 有明确商品图，想快速出片 | 上传图片 → AI 自动分析推荐 → 选运动效果 → 生成，全程无文字输入 |
| 进阶优化 | 对效果不满意，想调细节 | 打开「高级设置」，AI 生成的 prompt 草稿可手动编辑，调整后重新生成 |
| 效果验证 | 想知道生成的视频质量如何 | 结果页展示质检评分卡，各维度分数 + 综合评级 + 是否可投放 |

## 4. 功能范围

### V1 (MVP — 当前)

- [x] 图片上传 + AI 自动分析（Mock 推导）
- [x] 6 种运动效果可视化选择
- [x] 投放场景选择（开屏/头图/信息流）
- [x] 生成进度动效
- [x] 结果页（视频预览 + 质检评分 + 做同款）
- [x] 模板市场（6 个预设模板）
- [x] 效果看板（趋势图 + 历史记录）
- [x] 高级设置（手动调整 prompt，折叠面板）

### V2（规划中）

- [ ] 真实 AI 视频生成 API 接入
- [ ] 图片质量前端预检（尺寸、清晰度、主体占比）
- [ ] 多模型并行生成 + 自动选择最佳结果
- [ ] 用户反馈闭环（评分 → 偏好学习 → 推荐优化）

### 不做的

- 文生视频（目标用户没有视频创意能力，图生视频是正确起点）
- 视频编辑/剪辑（超出 MVP 范围，且增加产品复杂度）
- 品牌客户的定制化工作流（V1 聚焦中小广告主）

## 5. 用户流程

```
首页 → [开始生成] → 上传图片 → AI 分析 → 选择场景 + 运动效果
                                    │
                    模板市场 → [做同款] ─┘
                                    │
                              [开始生成]
                                    │
                           生成进度动效（6-8s）
                                    │
                              结果页 → 预览视频
                              │    ├→ 查看质检评分
                              │    ├→ 下载
                              │    └→ [做同款] 返回创建页
                              └→ 效果看板
```
