# 关键产品决策记录

本文档记录 VidMotion 项目中的核心产品决策及其背后的逻辑。这些决策反映了 AI PM 在产品设计中需要权衡的真实 trade-off。

---

## 决策 1：为什么选择「图片微动」而非「文生视频」？

**2023 年（原始项目背景）选择 AnimateDiff 做图生微动的理由：**

1. **模型能力边界**：2023 年中，AnimateDiff 在图生视频（尤其是保持主体一致性）方面表现稳定，而同期 SVD 和早期 Sora 在主体一致性上波动较大
2. **用户素材约束**：中小广告主手上只有商品图片，没有视频素材。图生视频是自然起点
3. **质量可控性**：微动（镜头缓推/视差/浮动）的运动范围小，画面主体不变形，适合品牌安全的广告场景

**如果 2026 年重新选择：**
- 可灵（Kling）、Luma Dream Machine 在文生视频和图生视频上已大幅进步
- 但「微动」作为产品形态依然成立——不是因为技术限制，而是因为广告场景不需要复杂的叙事性视频。开屏/Banner/信息流的核心需求是**在 3-5 秒内吸引注意力**，而非讲述故事
- 技术选型应从市场最新可用模型中评估，但产品形态（短时微动）保持不变

**产品启示**：技术选型受时间点约束，但产品形态应由用户场景定义。面试中容易被追问「如果今天重新选会选什么」，好的回答是在承认技术进步的同时，解释产品形态的稳定性来自场景理解。

## 决策 2：为什么不做传统 Prompt 文本框，而是用可视化效果卡片？

**常见做法**：Runway、Pika 等工具提供一个大文本框，用户输入 prompt。

**我们的选择**：用 6 张可视化效果卡片替代 prompt 输入。

**理由**：

1. **用户画像匹配**：中小店主不是创作者，不会写 prompt。「镜头缓推」比「camera slowly zooming in with Ken Burns effect」更可理解
2. **试错成本差异**：文生图一次生成 4 张、几秒出结果，试错代价低；视频生成耗时几十秒到几分钟、单次消耗大量 token，试错代价高得多
3. **运动效果是可枚举的**：与「画面中有哪些元素」不同，广告场景中的镜头运动类型是有限的（几种到十几种），天然适合用预设 + 可视化选择而非自由文本
4. **但保留灵活出口**：「高级设置」折叠面板中保留了 prompt 文本框（初始为空，显示 AI 自动生成的 prompt），进阶用户可以手动调

**代价**：限制了创意自由度。对于专业创作者，预设效果可能不够用。但这是一个有意识的 trade-off——先服务 80% 的中小广告主，而非 20% 的专业用户。

## 决策 3：「做同款」为什么是核心功能？

**表面逻辑**：降低新用户的决策门槛——不知道选什么风格时，直接复制成功案例的配置。

**深层逻辑**：做同款是**增长飞轮的起点**：
1. 用户 A 用模板生成视频 → 投放 → 产生 CTR 数据
2. 高 CTR 的视频被收录为模板 → 展示在模板市场
3. 用户 B 看到「CTR +18%」→ 点击做同款 → 换图生成 → 投放 → 产生数据
4. 循环往复，模板库越来越丰富，每个模板背后都有真实投放数据背书

**为什么不担心同质化**：
- 商品图不同（用户 A 卖运动鞋、用户 B 卖保温杯），最终视频画面完全不同
- 模板只锁定「运动方式 + 镜头节奏 + 节奏感」，不锁定画面内容
- 类似于「用同一个 Instagram 滤镜」不会让你的照片和别人一样

**需要注意的问题**：
- 如果某个模板被过度使用，同一广告位可能出现「运动方式撞车」。需要设置模板热度上限或引入运动方式的微变体

## 决策 4：质检评分为什么包含这三个维度？

| 维度 | 为什么重要 | 对应的用户感知 |
|------|-----------|---------------|
| 内容准确性 | 主体识别错误 → 视频展示的不是用户的产品 → 完全不可用 | 「这视频里是我的产品吗？」 |
| 动作连贯性 | 闪烁/跳帧/画面撕裂 → 看起来廉价 → 用户不敢投放 | 「这视频看起来像盗版的吗？」 |
| 色彩一致性 | 色调偏移 → 产品颜色失真 → 误导消费者 | 「我的红色裙子看起来像橙色的」 |

**为什么没有「美学评分」**：美学过于主观，而且是相对的——不同行业、不同品牌调性对「美」的定义不同。质检应该检验「不出错」，而非「有多好看」。CTR 数据才是衡量「好看」的最终标准。

**阈值设置逻辑**：综合 ≥ 80 可投放。单维度 ≥ 85/80/85。阈值不是一次拍板的——原项目通过「驱动运营、测试部门产出测评结果」来校准，确保 80% 人工测评通过率对应综合 80 分的阈值线。

## 决策 5：为什么先不做品牌客户？

品牌客户的核心诉求是「创意内容不能增删改」——而 AI 视频生成本质上是「对原素材进行了创造性的修改」。这两个需求在根本上冲突。

**可行的解法**（但超出 V1 范围）：
- 针对品牌客户，只开放「轻微」强度的运动效果（镜头缓推）
- 提供逐帧审核功能，品牌方可以标记任何「越界」的帧
- 与品牌客户共建风格指南（brand style guide 映射到 prompt 约束）

V1 选择聚焦中小广告主，不是因为品牌客户不重要，而是因为中小广告主的需求更清晰、决策路径更短、产品价值更容易验证。

---

*这些决策记录的目的不是证明「我们是对的」，而是展示在约束条件下如何做出有意识的权衡——这正是 PM 的核心能力。*
